# tslint-config-m18

[M18](https://github.com/m18ru) config for [TSLint](https://github.com/palantir/tslint).

## Installation

```
npm install --save-dev tslint-config-m18
```

## Usage

In `tslint.json`:

```json
{
	"extends": "tslint-config-m18"
}
```
